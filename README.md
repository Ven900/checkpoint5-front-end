## 📚 Sobre o Projeto

Este projeto foi desenvolvido para o **Checkpoint 5 da disciplina de Front-end Design (FIAP)**.

O objetivo é criar uma **landing page única, totalmente responsiva**, contendo múltiplas seções com navegação fluida, interatividade e aplicação de boas práticas de desenvolvimento front-end.

---

## 🎯 Objetivos do Projeto

* Construir uma página seguindo o layout proposto
* Garantir **responsividade completa (desktop, tablet e mobile)**
* Aplicar conceitos de:

  * HTML semântico
  * CSS (layout, animações, responsividade)
  * JavaScript (interações e manipulação do DOM)

---

## 🧩 Estrutura da Página

### 1️⃣ Início

* Menu fixo com navegação suave (scroll)
* Seção principal com botão “Start”

### 2️⃣ Integrantes

* Lista com nomes e RMs
* Interação visual ao passar o mouse (hover)

### 3️⃣ Pokémons

* Filtro por categorias (água, fogo, planta, etc.)
* Exibição dinâmica dos personagens
* Modal com detalhes ao clicar em um Pokémon

### 4️⃣ Cartas

* Formulário para criação de carta personalizada
* Dados exibidos dinamicamente na interface

### 5️⃣ Footer

* Informações institucionais
* Layout organizado e espaçado
